# Homebrew Tap for WAFtester

[![Latest Version](https://img.shields.io/github/v/release/waftester/waftester?label=version)](https://github.com/waftester/waftester/releases/latest)

Official [Homebrew](https://brew.sh) tap for [WAFtester](https://waftester.com) — the WAF security testing CLI.

Detect which WAF protects a target, benchmark its rule coverage across OWASP categories, and test bypass techniques — all from a single binary.

## Install

```bash
brew tap waftester/tap
brew install waftester
```

## Upgrade

```bash
brew upgrade waftester
```

## Platforms

| OS    | Architecture |
|-------|-------------|
| macOS | Intel (x86_64), Apple Silicon (arm64) |
| Linux | x86_64, arm64 |

## How updates work

This formula is automatically updated by [GoReleaser](https://goreleaser.com) when a new version of WAFtester is released. No manual maintenance required.
